# skypro-wallet 
+ Пользователь может зарегистрироваться и войти в кошелёк
+ Авторизованный пользователь может добавить новый расход, выбрав категорию, дату и сумму
+ Авторизованный пользователь может провести анализ расходов за выбранный период

## Recommended IDE Setup

[VS Code](https://code.visualstudio.com/) + [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).

- Firefox:
  - [Vue.js devtools](https://addons.mozilla.org/en-US/firefox/addon/vue-js-devtools/)
